# Exempt from Recording Settings

**Source:** https://www.ikuai8.com/support/ymgn/lyym/behavior-control/feqef/ip.html

## I. Overview

Exempt from Recording Settings allows administrators to exclude specific IP or MAC addresses from behavior monitoring. When behavior recording is enabled, any device added to this exemption list will **not** have its activities logged, including:

- Website browsing history
- IM communications
- Device login/logout events

---

## II. How to Use

Add IP addresses or MAC addresses to the exemption list. Any entry added here is excluded from all behavioral tracking, regardless of what is configured in the general Behavior Record Settings.

> This setting works in conjunction with [Behavior Record Settings](../behavior-record-settings.md). Exemptions only take effect when behavior recording is active.
