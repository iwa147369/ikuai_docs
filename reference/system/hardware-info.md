## III. Reference Information

### Device Capacity (Recommended Terminal Count)

The number of manageable terminals primarily depends on CPU type. The values below are for reference only — actual capacity varies by usage environment and bandwidth.

![Capacity Chart](https://www.ikuai8.com/attached/image/20150405/20150405144743_82629.png)

> **Note:** Capacity varies by network environment and bandwidth requirements.

### Memory Requirements

- Minimum: **1 GB** RAM
- 32-bit system: maximum recognized memory is **4 GB** (typically shows ~3 GB)
- 64-bit system: requires **4 GB or more** RAM

### Disk Requirements

- Minimum: **1 GB** disk space

### NIC Requirements

- Recommended: Intel-chipset discrete NICs
- Wireless NICs: **not supported**
- USB wired NICs: supported drivers include RTL8150/8152/8153/8156, AX88179/8817x, AQC111
