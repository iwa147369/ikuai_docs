# Current Routing Table

**Source:** https://www.ikuai8.com/support/ymgn/lyym/wlsz/94829/5dc33.html

## I. Overview

The Current Routing Table page displays the router's active routing table in real time.

---

## II. Column Descriptions

| Column | Description |
|---|---|
| Line | The WAN circuit on which this routing rule applies |
| Destination Address | The network segment being accessed |
| Subnet Mask | Mask for the destination segment |
| Gateway | Next hop address; `0.0.0.0` indicates a directly connected route |
| Priority | Policy priority level |

---

## III. Version Notes

- IPv4 routing table viewing is available in all versions.
- IPv6 routing table viewing requires firmware version **3.7.16 or later**.
