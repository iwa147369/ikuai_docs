# WiFi Settings

**Source:** https://www.ikuai8.com/support/ymgn/lyym/wlsz/wifi.html

## I. Overview

The WiFi Settings page configures the built-in wireless radio of iKuai routers, including SSID names, passwords, and radio parameters.

---

## II. Configuration Modes

| Mode | Description |
|---|---|
| Standard Mode | Separate configuration for 2.4G and 5G radios |
| Dual-Band Unified Mode | Both bands share the same SSID and settings |

---

## III. Configuration Parameters

| Parameter | Description |
|---|---|
| SSID Name | Wireless network broadcast name |
| Security Type | Encryption method (e.g., WPA2, WPA3) |
| Password | Network access credential |
| Network Card Binding | Which LAN interface the wireless connects to (default: LAN1) |
| BSSID | Modify the wireless radio's MAC address |
| SSID Visibility | Hide the SSID so it does not appear in scan results |
| Guest Mode | Restricts clients from communicating with each other and from accessing the wired LAN |
