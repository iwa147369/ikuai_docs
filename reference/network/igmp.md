# IGMP Proxy

**Source:** https://www.ikuai8.com/support/ymgn/lyym/wlsz/igmp.html

## I. Overview

IGMP (Internet Group Management Protocol) is a multicast protocol that operates between hosts and multicast routers. iKuai supports **IGMPv2** and **IGMPv3**, primarily used for hotel IPTV services.

---

## II. Configuration

| Field | Description |
|---|---|
| Protocol Version | IGMPv2 or IGMPv3 |
| Upstream Port | The interface receiving the multicast signal source |
| Downstream Port | The LAN interface(s) delivering multicast to clients (multiple LAN ports supported) |
