# Push Notification Log (推送通知日志)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/rzzx/gegge/herr.html

## I. Overview

Shows successful push notification events from the router, including real-time notifications, periodic notifications, and expiry push alerts.

![Push Notification Log](https://www.ikuai8.com/images/support/tstz.png)

---

## II. How to Use

Filter by notification type (all / real-time / periodic / expiry). Enter a specific IP to see that device's notification history.

> **Note:** This log only shows **successfully delivered** push notifications.

---

## III. Log Retention

Max entries: **500**. See [System Log](../system-log.md) for the full retention table.
