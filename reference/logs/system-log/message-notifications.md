# Message Notifications

**Source:** https://www.ikuai8.com/support/ymgn/lyym/rzzx/xtrz/8eb96.html

## I. Overview

The Message Notifications page displays official announcements and updates pushed by iKuai to the router's management interface.

When iKuai releases new notifications — such as firmware announcements, security advisories, or feature updates — administrators can view the records here.

**Location:** Logs Center → System Logs → Message Notifications
