# Wireless Terminal Log (无线终端日志)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/rzzx/fe4g/htehb.html

## I. Overview

Shows connection details for wireless terminals on each AP.

![Wireless Terminal Log](https://www.ikuai8.com/images/wxzdrz.jpg)

---

## II. How to Use

View and filter by:
- AP MAC address
- Terminal wireless connection start and end times
- Terminal connection status

Export the wireless terminal internet access log.

> **Note:** Requires AP firmware version **1.1.2 or later**.

---

## III. Log Retention

Max entries: **10,000**. See [System Log](../system-log.md) for the full retention table.
