# Peripheral Device Monitoring

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ztjk/2020-09-08-02-28-12.html

## I. Overview

The Peripheral Device Monitoring page displays peripheral devices that have been successfully added to the iKuai Cloud platform, providing real-time status monitoring and remote access.

---

## II. Key Features

- Real-time device status monitoring
- Last online timestamp tracking
- Remote web interface access to individual devices

---

## III. Setup Steps

1. Log into the **iKuai Cloud** platform and open the **Network Inspection** module.
2. Navigate to **Device Management → Peripheral Devices**.
3. Add your devices following the peripheral device setup tutorial.
4. Devices will automatically appear in the router's **Status Monitoring → Peripheral Device Monitoring** dashboard.
