# Camera Monitoring

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ztjk/2022-12-22-09-20-50.html

## I. Overview

The Camera Monitoring page displays information about IP cameras connected to the local network, allowing administrators to access their management interfaces directly from the router dashboard.

---

## II. Requirements

| Requirement | Detail |
|---|---|
| Router firmware | Version 3.7.0 or higher |
| Supported hardware | x86 platform (including free versions), IK-R2001, IK-R2002, IK-A2003, IK-R2005, IK-A220PRO, IK-A320PRO |

---

## III. How to Use

Click the **Login** button next to a camera to open its web management interface through the router dashboard.

---

## IV. Important Limitations

- **Local access only** — The login function only works when accessing the router's web interface from within the local network. Remote access via WAN or the iKuai Cloud platform will not successfully open camera management pages.
- **Cloud platform** — The cloud platform supports one-click login to camera management pages but does **not** support live video viewing.
