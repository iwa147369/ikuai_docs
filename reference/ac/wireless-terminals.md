# Wireless Terminal List

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ac/dqwqd.html

## I. Overview

View information about all WiFi clients currently connected to any AP managed by the AC, including IP address, MAC address, signal strength, and traffic statistics.

![Wireless Terminal List](https://www.ikuai8.com/images/support_202109/AC3.png)

This page is read-only — it is for monitoring purposes only and has no configurable settings.

---

## II. Column Descriptions

| Column | Description |
|---|---|
| Terminal IP | The IP address of the connected client. May be blank if the client has had no traffic for an extended period — the router cannot identify it without traffic. |
| Terminal MAC | The MAC address of the connected client. |
| AP Info | The MAC address and remark (label) of the AP the client is connected to. |
| SSID | The wireless network name the client is connected to. |
| Signal Strength (dBm) | The signal level as measured by the AP. This value is also used for the "minimum RSSI kick" threshold if configured. |
| Connection Duration | How long the client has been connected to this AP. |
| Upload Rate | The client's current real-time upload speed. |
| Download Rate | The client's current real-time download speed. |
| Remark | The label associated with this terminal. |
