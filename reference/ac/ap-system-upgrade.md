# AP System Upgrade

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ac/3212.html

## I. Overview

The AP System Upgrade page lets you check the current firmware version of each AP and upgrade them to a newer version when one is available.

---

## II. How to Upgrade

When a newer AP firmware version is available, the system displays the latest version number in the AP list.

You can upgrade APs using either method:

1. **Single upgrade** — Click the upgrade button in the Actions column of the target AP.
2. **Batch upgrade** — Select multiple APs, then click the **Batch Upgrade** button to upgrade them all at once.

---

## III. Important Precautions

- Do not power off or perform any operations on the AP during the upgrade process.
- Incorrect DNS settings can cause AP upgrade failures. Ensure DNS is configured correctly before upgrading.
