# Restart / Shutdown (重启关机)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/xtsz/sdqf.html

## I. Overview

The Restart/Shutdown page supports both immediate and scheduled restart or shutdown operations.

> **Note:** Before configuring scheduled restart/shutdown tasks, ensure the router's system time is accurate or that the NTP service is enabled.

---

## II. Configuration

### Immediate Restart / Shutdown

![Immediate Restart/Shutdown](https://www.ikuai8.com/images/support/cqgj.png)

Clicking the button immediately restarts or shuts down the router.

### Scheduled Events

![Scheduled Events](https://www.ikuai8.com/images/support/cqgj1.png)

| Field | Description |
|---|---|
| Scheduled Event | Select **Restart** or **Shutdown**. |
| Recurrence | **Once** — runs once on a specific date. **Daily** — runs every day at the specified time. **Weekly** — runs on a specified day of the week. **Monthly** — runs on a specified day of the month. |
| Date | The specific date or day of the week to run the event. |
| Time | The time (24-hour format) to execute the restart or shutdown. |
| Remarks | An optional note for this event. |
| Status | Enable or disable this scheduled event. |
| Actions | Edit, disable, or delete the event. |
