# File Management (文件管理)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/xtsz/2019-12-13-02-14-28/2019-12-13-02-23-05.html

## I. Overview

Browse, upload, and manage files on the router's disk partitions directly through the web interface.

---

## II. Interface Reference

| Control | Description |
|---|---|
| Upload | Upload a file to the current directory. |
| New Folder | Create a new folder in the current directory. |
| Cut | Select files and click Cut to move them to another directory. Cut only works within the same disk. |
| Delete | Delete the selected files or folders. |
| Navigation Bar | Browse directories using the breadcrumb bar, or click **Go Up** to navigate to the parent directory. |
| File Name | Name of the file or folder. |
| Size | Size of the file or folder. |
| Type | `Directory` for folders, `File` for files. |
| Modified Time | Last modification timestamp. |
| View Path | Shows the full path of the selected file or folder. Includes a one-click copy button — useful for referencing paths in VM configurations. |
| Rename | Rename a file or folder. |
| Delete | Delete a file or folder. |

---

## III. Important Notes

1. Files can only be uploaded to **General Storage** partitions — not to system or service-specific partitions.

---

## IV. Usage Example

1. Navigate to the desired disk directory.
2. Click **Upload**, select a local file in the dialog, and confirm to begin the upload.
