# Application Binding

**Source:** https://www.ikuai8.com/support/ymgn/lyym/xtsz/2019-10-18-02-59-36.html

## I. Overview

Application Binding integrates the iKuai router with DingTalk or WeChat Work, enabling:

- **DingTalk Flash Transfer** — Fast file transfer via DingTalk
- **One-click network connection** — Users connect to the network directly through DingTalk or WeChat Work

---

## II. Setup

Scan the QR code displayed on the Application Binding page using DingTalk or WeChat Work with an **administrator account** to complete binding.

> DingTalk binding requires an account with admin privileges to ensure access to team selection.

---

## III. Important Notes

| Item | Detail |
|---|---|
| DingTalk exclusive passwords | Can be distributed to up to 300 devices |
| Password expiry | Credentials do not expire, but become invalid if reset |
| Bypass mode | Cannot retrieve terminal data; the router must operate as primary gateway |
| WeChat Work on Android 10+ | One-click network connection has known issues on Android 10 and above. iOS devices function normally. |
