# AC Management Demo

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ac/ac.html

## I. Overview

This page demonstrates the core AC (Access Controller) management features available in the iKuai router interface for monitoring and controlling wireless networks.

---

## II. Key Features

### Real-time Monitoring

Navigate to the **Wireless Overview** page to view:

- AP status
- Number of connected terminals
- Network quality scores
- Traffic statistics
- Transmission quality metrics

### AP Details and Configuration

The AP List page provides comprehensive AP information and configuration options, including:

- Device status
- Basic hardware information
- Per-AP state data

### Terminal Management

The Wireless Terminal List page displays all connected wireless devices. Selecting a terminal shows its basic information and current connection status.

### Connection Logs

The system records terminal connection and disconnection events, allowing administrators to review access history.
