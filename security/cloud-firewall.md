# Cloud Firewall

**Source:** https://www.ikuai8.com/support/ymgn/lyym/aqsz/2024-03-12-07-42-40.html

## I. Overview

Cloud Firewall is a next-generation firewall service that integrates with the iKuai routing gateway to provide comprehensive traffic management and network security protection.

**Key capabilities:**
- Threat intelligence
- IPS (Intrusion Prevention System)
- Antivirus protection
- Malicious URL filtering
- Behavior management
- Audit logging with full traffic visibility

---

## II. Requirements

| Requirement | Detail |
|---|---|
| System version | 3.7.12 or higher |
| Architecture | x86 64-bit routers only |
| CPU | Dual-core minimum |
| Available RAM | 2.3 GB minimum |

---

## III. Activation

1. Log in to the **iKuai Cloud Platform** and open the **Network Inspection** module.
2. Select the target router.
3. Purchase a service duration and complete payment.

---

## IV. Management

After activation, navigate to **Security Settings → Cloud Firewall** in the router dashboard to:

- View router ID, serial number, version, and validity period
- Renew the service
- Adjust protection settings
- Uninstall the service
