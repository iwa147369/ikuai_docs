# Block Entertainment Websites (禁止娱乐网站)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/behavior-control/mjtt/mtty5.html

## I. Overview

One-click blocking by website category using iKuai's built-in URL database. No need to enter individual domains manually.

---

## II. Field Reference

| Field | Description |
|---|---|
| Website Type | Category to block (e.g., gaming, video, social media). |
| Internal IP | The LAN client IP(s) this rule applies to. |
| Schedule | The recurrence period for this rule. |

---

## III. Notes

- Do **not** combine this feature with **URL Whitelist** mode. Whitelist mode blocks all unlisted domains by default, which conflicts with category-based blocking logic.
- The URL database is maintained by iKuai. Use the **Custom URL Library** page to query or submit missing domains.
