# Custom URL Library (自定义网址库)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/behavior-control/mjtt/htrhr.html

## I. Overview

Query the built-in iKuai URL database and submit missing URLs to be added.

---

## II. Usage

1. Enter a domain name to search the database.
2. If the domain is not found, submit it with a category and type for inclusion.

### Match Types

| Type | Description |
|---|---|
| Fuzzy | `baidu.com` matches `*.baidu.com` (all subdomains) |
| Exact | `www.baidu.com` matches only that exact domain |

---

## III. Notes

- This database is used by **Block Entertainment Websites** and related URL-based controls.
- Submitted entries are reviewed by iKuai before being added to the official database.
