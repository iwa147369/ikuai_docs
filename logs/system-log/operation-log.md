# Operation Log (操作日志)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/rzzx/xtrz/603b2.html

## I. Overview

Records all administrative actions taken after logging into the router, including timestamp, username, IP address, function area, and event description.

---

## II. Common Issues

### Why does the log show "User: System"?

Entries with username **System** are automatically generated by the router itself, not by a human administrator. Examples:
1. When web authentication is enabled on the iKuai Cloud platform, it is automatically pushed to the router and activates web authentication in the router's config.
2. When an AP comes online, the router automatically applies default configuration to it and registers it in the AC management section.

---

## III. Log Retention

See [System Log](../xtrz.md) for the full retention table. Operation Log max entries: **5,000**.
