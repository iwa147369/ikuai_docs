# Dynamic DNS Log (动态域名日志)

**Source:** https://www.ikuai8.com/support/ymgn/lyym/rzzx/gegge/srrge.html

## I. Overview

Shows update results for configured dynamic DNS (DDNS) entries.

![Dynamic DNS Log](https://www.ikuai8.com/images/support/dtym.png)

---

## II. How to Use

Entries only appear when at least one DDNS rule is configured under **Advanced Applications → Dynamic DNS**.

Each entry shows whether the DDNS update succeeded or failed, along with the updated domain and IP address.

Clear the log using the Clear button in the top-right corner.

---

## III. Log Retention

Max entries: **5,000**. See [System Log](../xtrz.md) for the full retention table.
