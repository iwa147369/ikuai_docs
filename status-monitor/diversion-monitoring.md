# Traffic Diversion Monitoring

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ztjk/ed49c.html

## I. Overview

The Traffic Diversion Monitoring page displays the different traffic diversion methods in use and the number of connections successfully diverted through each method.

---

## II. Page Controls

| Button | Description |
|---|---|
| Clear All Data | Resets all statistical values in the table to zero |
| Refresh | Manually refresh data, or enable auto-refresh |

---

## III. Notes

- After a router initialization (factory reset), the diversion statistics on this page do **not** automatically reset to zero. The data persists until manually cleared.
