# Switch Monitoring

**Source:** https://www.ikuai8.com/support/ymgn/lyym/ztjk/jhjjk.html

## I. Overview

The Switch Monitoring page provides SNMP-based monitoring of internal network switches. It displays switches that have been successfully added to the iKuai Cloud platform.

---

## II. Setup Steps

1. Log into the **iKuai Cloud** platform and open the **Network Inspection** module.
2. Navigate to **Device Management → Switch**.
3. Add your switch devices following the iKuai Cloud switch configuration tutorial.
4. Added switches will automatically appear in **Router Dashboard → Status Monitoring → Switch Monitoring**.
